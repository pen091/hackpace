Windows Installation Guide

Tor Browser is based on Mozilla Firefox's ESR (Extended Support Release), which periodically updates to include critical security updates from Firefox's main version. [citation:1]

Installation Steps:

  1. Navigate to the Tor Browser download page. [citation:1]
  2. Download the Windows .exe file. [citation:1]
  3. (Recommended) Verify the file's signature. [citation:1]
  4. When the download is complete, double click the .exe file. [citation:1]
  5. Complete the installation wizard process. [citation:1]
Note:

Support for Windows 7, 8, and 8.1 is discontinued with Tor Browser 14. Users on these legacy operating systems will continue to receive security updates for Tor Browser 13.5 until at least September 2025. [citation:1]

Linux Installation Guide

Tor Browser is supported on any modern Linux-based operating system. [citation:1]

Installation Steps:

  1. Navigate to the Tor Browser download page. [citation:1]
  2. Download the Linux .tar.xz file. [citation:1]
  3. (Recommended) Verify the file's signature. [citation:6]
  4. Extract the archive using an archive manager or the command line. [citation:1]
  5. Navigate to the Tor Browser directory and run ./start-tor-browser.desktop. [citation:1]
Command Line Method:

After downloading, extract the archive with: tar -xf [TB archive] [citation:1]

From inside the Tor Browser directory, launch with: ./start-tor-browser.desktop [citation:1]

If this fails, make the file executable: chmod +x start-tor-browser.desktop [citation:1]

Android Installation Guide

Tor Browser for Android requires Android 5.0 or later. [citation:10]

Installation Steps:

  1. Visit the official Tor Project website or Google Play Store.
  2. Download the Tor Browser for Android APK file.
  3. Enable installation from unknown sources in your Android settings.
  4. Open the downloaded APK file and complete the installation.
  5. Launch Tor Browser and connect to the Tor network.
Note:

For maximum security, download Tor Browser only from official sources like the Tor Project website or official app stores. [citation:3]
